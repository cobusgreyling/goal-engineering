## Categories

| Category | Use for |
|----------|---------|
| **Pattern requests** | New patterns, verifier tweaks |
| **Story submissions** | Real runs — wins and failures |
| **Tooling & audit scores** | CLI improvements, G0–G3 debates |
| **Goal-ready reviews** | "Is my GOAL.md ready for G3?" |
| **General** | Questions and links |

Category config: [.github/DISCUSSION_TEMPLATE/categories.yml](../.github/DISCUSSION_TEMPLATE/categories.yml)

## Submit a story

1. Copy `stories/ci-red-to-green.md` format
2. Include: pattern, G-level, turns, verifier REJECT/PASS
3. Open PR adding row to `stories/README.md`

Honest failures encouraged — see [mega-goal-failure.md](../stories/mega-goal-failure.md).
